# Generalised Profiling Tutorial
This repository contains two Jupyter Notebooks that illustrate the use of generalised profiling as a method of inference for systems of ordinary differential equations (ODEs). These codes were written by Alexander Johnston. 
Two ODE models are examined: the classic damped, driven harmonic oscillator, and the Lotka-Volterra model of predator-prey dynamics. The codes were used to generate all figures and results in the paper "Efficient inference for differential equation models without numerical solvers", currently available as an arXiv preprint at.
