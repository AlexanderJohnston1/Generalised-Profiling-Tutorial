# Generalised Profiling Tutorial
This repository contains two Jupyter Notebooks that illustrate the use of generalised profiling as a method of inference for systems of ordinary differential equations (ODEs). These codes were written by Alexander Johnston. 
Two ODE models are examined: the classic damped, driven harmonic oscillator, and the Lotka-Volterra model of predator-prey dynamics. The codes are written in Julia and were used to generate all figures and results in the paper "Efficient inference for differential equation models without numerical solvers", currently available as an arXiv preprint at https://arxiv.org/abs/2411.10494.

Each folder contains a code that generates new synthetic data and then applies the generalised profiling method ("Harmonic Oscillator.ipynb" and "Lotka-Volterra.ipynb", respectively). In addition, each folder also contains another code ("Harmonic Oscillator (data set used in paper).ipynb" and "Lotka-Volterra (data set used in paper).ipynb", respectively) which, along with the data sets in each folder, allow the user to reproduce the plots produced for the arXiv preprint.
